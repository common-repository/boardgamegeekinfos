Short code for embedding game infos from Board Game Geek

== Description ==

Add short code [bgg id=GAMEID] for embedding game infos from Board Game Geek inside your content.

This plugin use the [BGG XML API](http://www.boardgamegeek.com/xmlapi?).
[Read terms and conditions](http://boardgamegeek.com/wiki/page/XML_API_Terms_of_Use?)

Thanks to [Andrea "Liga" Ligabue](http://ludologo.com)

Released under [MIT License](http://en.wikipedia.org/wiki/MIT_License).

== Installation ==

1. Upload `boardgamegeekinfos` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Insert shortcode [bgg id=GAMEID] inside your content. Grab ID game from game url on boardgamegeek.com. Example: http://boardgamegeek.com/boardgame/94731/heroica-fortaan id is 94731

== Frequently Asked Questions ==

= Where I can get game code?  =

Open game page on boardgamegeek.com, and get the code from url.
Example: http://boardgamegeek.com/boardgame/94731/heroica-fortaan

Game ID is 94731
